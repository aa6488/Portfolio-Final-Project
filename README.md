## 프로젝트 소개

- COCO 프로젝트는 자신의 아이디어를 구상하고 팀원을 모아 서로가 협동 및 협력하여 Project를 만드는
  개발자들의 일정관리 및 협업 환경을 제공해주는 서비스와 더불어 개발자들의 정보 공유 게시판 및 프로젝트 추천, 자유 게시글을 만들 수 있는 웹페이지 프로젝트 입니다.


  
- 진행기간 : 2024년 01월 18일 → 2024년 02월 26일 의 기간동안 진행했습니다.
